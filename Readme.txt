Данное расширение автоматически добавляет третий вид категории, прайслист(Таблицы).

Не трогает файлы магазина.

Работает из коробки без вмешательства только на дефолтных темах!!! 
К остальным темам и для нормальной работы с фильтрами требуется адаптация JS скрипта!!!

Язык: ru.

Тестировалось на Opencart 2.0.X и Opencart 2.1.X , Ocstore 2.0.X, и Ocstore 2.1.X

Демо на сайте: http://svg.proitservice.in.ua/

Установка:

Через меню "Модули" -> "Установка расширений" установить файл "priceview.ocmod.zip".

Обновить кэш модификаторов "Модули" -> "Модификаторы" Нажав на кнопку "Обновить"
